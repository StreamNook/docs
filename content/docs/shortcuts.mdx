---
title: Keyboard Shortcuts
description: Every keyboard shortcut in StreamNook, grouped by area.
---

StreamNook is built to drive from the keyboard. Every shortcut below can be changed in **Settings, Keybindings**, except the reserved ones noted at the end.

## Application

| Action | Key |
| --- | --- |
| Open command palette | `Ctrl+K` |
| Open settings | `Ctrl+,` |
| Show keyboard shortcuts | `Shift+/` |
| Open changelog | `Ctrl+Shift+C` |

## Navigation

| Action | Key |
| --- | --- |
| Go home | `Alt+Home` |
| Open Drops | `Ctrl+Shift+D` |
| Open Badges and Paints | `Ctrl+Shift+B` |
| Open Whispers | `Ctrl+Shift+W` |
| Refresh followed streams | `Ctrl+Alt+R` |
| Next followed stream | `Alt+N` |
| Previous followed stream | `Alt+P` |

## Player

| Action | Key |
| --- | --- |
| Play / pause | `Space` or `K` |
| Mute / unmute | `M` |
| Volume up / down | `Up` / `Down` |
| Seek back / forward 10s | `Left` / `Right` |
| Fullscreen | `F` |
| Theater mode | `T` |
| Picture-in-picture | `P` |
| Speed up / down | `Shift+.` / `Shift+,` |
| Create clip | `Alt+X` |
| Restart stream | `R` |
| Close stream | `Shift+Q` |

## Moderation

Focus a message first, then act on it. Available to mods and broadcasters.

| Action | Key |
| --- | --- |
| Focus newer / older message | `J` / `K` |
| Open focused user's card | `U` |
| Delete focused message | `Delete` |
| Timeout (purge / 10m / 1h) | `1` / `2` / `3` |
| Ban / unban | `B` / `A` |
| Clear focus | `Esc` |

## Chat input

These are fixed and not rebindable.

| Action | Key |
| --- | --- |
| Send message | `Enter` |
| Send and keep text | `Ctrl+Enter` |
| New line | `Shift+Enter` |
| Emote and name autocomplete | `Tab` |

## Multi-chat

Fixed shortcuts for the multi-chat window.

| Action | Key |
| --- | --- |
| New chat tab | `Ctrl+T` |
| Close chat tab | `Ctrl+W` |
| Next / previous tab | `Ctrl+Tab` / `Ctrl+Shift+Tab` |
| Jump to tab 1 to 9 | `Ctrl+1` to `Ctrl+9` |

<Tip>
In **Settings, Keybindings** you can search any command, rebind it, and reset everything to defaults. The tab flags conflicts and lists keys it cannot reassign.
</Tip>
