title: Command Palette
description: Run any action, jump anywhere, and paste snippets from one search box.
---

The command palette is a single search box that runs nearly anything in StreamNook: app actions, navigation, settings, channel search, player controls, and a library of chat snippets. Press `Ctrl+K` (`Cmd+K` on macOS) to open it from anywhere, including multi-chat popout windows.

## Opening and navigating

| Key | Action |
| --- | --- |
| `Ctrl+K` / `Cmd+K` | Open or close the palette. |
| Up / Down | Move the selection. |
| Home / End | Jump to the first or last result. |
| Enter | Run the selected row. |
| Esc | Close. |

Start typing to search everything at once. With the box empty, the palette shows your recent commands, actions for whatever you are currently watching, and quick jump-to destinations.

## What you can do from it

The palette pulls from many sources, so one search covers a lot:

- **Quick actions:** open Drops, Badges and Paints, Whispers, the Marketplace, your profile, settings, or What's New; go home; refresh your follows; watch a random or your top live follow; set a sleep timer (15m, 30m, 1h, 2h) or cancel it; open the GitHub repo or report an issue.
- **Current stream** (while watching): play or pause, mute, fullscreen, picture-in-picture, volume, theater mode, restart or stop the stream, follow or favorite the channel, create a clip, watch the latest VOD or top clip, pop the chat out, view drops for the game, or open the streamer's profile.
- **Share** (while watching): copy the stream URL, a markdown link, share text, or an embed snippet, or open the stream or schedule on twitch.tv.
- **Settings:** search any settings tab or section by name and jump straight to it.
- **Channels and categories:** type a streamer or game name to search Twitch live, jump to a followed channel, browse a category, or open whispers with a recent chatter.
- **Snippets:** find and copy any chat snippet (see below).
- **Bindable actions** show their keyboard shortcut on the right, so the palette doubles as a way to learn shortcuts.

Selecting a row runs it immediately. Focusing a streamer row shows a short bio and their links underneath.

## Snippets

Snippets are reusable chat lines (copypastas and command templates). Selecting one copies it to your clipboard so you can paste it into chat. StreamNook ships a library grouped into categories like Classic, Hype, Reaction, F / RIP, Forsen Lore, Long Classic, Meme, and Chat Commands.

Manage them in **Settings, Command Palette, Snippet Manager**:

- **Search and filter** by category or favorites.
- **Favorite** a snippet with the star so it rises to the top of palette results.
- **Alias** a snippet with a short shortcut; typing the alias in the palette jumps straight to it.
- **Add, edit, and delete** your own custom snippets in any category. Built-in snippets can be favorited and aliased but not edited or deleted.

Your favorites, aliases, and custom snippets sync across all StreamNook windows.

## Plugins

Plugins can add their own rows to the palette, so an installed add-on's actions are searchable right alongside the built-ins. See [UI plugins](/docs/plugin-ui).
